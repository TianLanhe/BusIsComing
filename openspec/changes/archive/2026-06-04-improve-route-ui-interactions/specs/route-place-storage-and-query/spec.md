## MODIFIED Requirements

### Requirement: 界面只展示地点名称

系统 SHALL 在主界面和路线管理页展示路线时只显示起点和终点地点名称，不展示经纬度。

#### Scenario: 主界面折叠态展示路线选择项
- **WHEN** 主界面展示已选中的保存路线
- **THEN** 路线选择框折叠态第一行显示路线名称，第二行显示 `起点地点名称 -> 终点地点名称`

#### Scenario: 主界面下拉列表保持当前展示方式
- **WHEN** 用户展开主界面的路线选择下拉列表
- **THEN** 下拉列表继续按当前单项样式展示已保存路线，不改为主标题和副标题列表

#### Scenario: 路线管理页展示路线项
- **WHEN** 路线管理页展示已保存路线
- **THEN** 路线项显示路线名称以及 `起点地点名称 -> 终点地点名称`
